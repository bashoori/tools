## 🧩 About

**BitaDigital** is a creative micro-studio focused on building lightweight, browser-based tools that simplify data work.  
Each app is built using **Python + Streamlit** and designed to be:
- Fast ⚡  
- Intuitive 🧠  
- 100% Local 🔒  

Explore apps that help analysts, engineers, and students clean, transform, and visualize data — without writing a single line of code.
